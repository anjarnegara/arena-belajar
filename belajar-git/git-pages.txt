                     Fitur Github pages

hanya bisa untuk tampilan web static, mencakup html, css, js

tidak menyediakan Sql dan web apache server



buat repo terlebih dahulu,
dengan format nama
namaakun.githhub.io

contoh: anjar.github.io

masuk ke folder lokal, klik kanan pakai git bash
jadiin repo dulu, git init
masukin repo ke repo github.io, git remote add origin https://github.com/namaakun/anjar.github.io.git

commit dulu ya..
git add .
git commit -m "  "


lalu push folder lokal ke github, karena ini pertamakali pak -u
git push -u origin master(nama branch)
