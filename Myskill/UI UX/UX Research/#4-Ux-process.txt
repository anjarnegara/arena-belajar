various kind of design proses

1. design thinking
   
   empati   >   define  >   ideate  >   prototype     >   test

how..
empati = user interview, form survey, riset kompetitor
define = analisa riset, data clustering, double diamond
ideate = brainstorming, storyboard, scamper, lofi, crazy eight
prototype = hi fi, mockup prototype
test = usability testing



2. design sprint
   
   map      >      sketch   >   decide  >   prototype   >   test


   

3. lean ux

think   >   make     >       check


4. double diamond

explore problem |   decide what to fix        test
